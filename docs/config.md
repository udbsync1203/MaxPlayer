# Конфигурация

## Где хранится

Файл конфигурации создаётся автоматически в системной директории пользователя:

```
<UserConfigDir>/MaxPlayer/config.json
```

`UserConfigDir()` — стандартная директория Go для пользовательских конфигов.

## Формат

```json
{
  "musicFolder": "/absolute/path/to/music"
}
```

## Логика загрузки

- если файла нет — создаётся пустой конфиг
- путь нормализуется (trim + `filepath.Clean`)
- пустой путь допустим, но библиотека не сканируется

## API управления

Методы `SetMusicFolder` и `GetMusicFolder` описаны в `backend-api.md`.
