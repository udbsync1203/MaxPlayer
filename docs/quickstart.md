# Быстрый старт

## Требования

- Go 1.23
- Node.js + npm
- Wails CLI v2

## Установка зависимостей фронта

```bash
cd frontend
npm install
```

## Запуск в dev‑режиме

```bash
wails dev
```

Wails поднимет фронтенд‑сервер и откроет приложение.

## Сборка

```bash
wails build
```

Результаты сборки будут в стандартной папке Wails `build/`.

## Минимальная проверка работы

1. Запустите приложение.
2. В UI должна появиться строка `Hello from backend` — это ответ `Hello()`.
3. При интеграции реального UI используйте методы из `backend-api.md`.
